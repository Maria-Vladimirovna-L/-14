## Описание программы

«Book Tracker» — это простое и удобное графическое приложение для учёта прочитанных книг. Программа позволяет:
*   Добавлять новые книги с указанием названия, автора, жанра и количества страниц.
*   Хранить все данные локально в формате **JSON**.
*   Фильтровать список книг по жанру и минимальному количеству страниц.
*   Просматривать полный список добавленных книг в удобной таблице.

Приложение не требует подключения к интернету и написано на языке Python с использованием библиотеки `tkinter`.

## Системные требования

*   **Операционная система:** Windows, macOS или Linux.
*   **Python:** версия 3.8 или новее.
*   **Библиотеки:** `tkinter` (входит в стандартную библиотеку Python), `json` (входит в стандартную библиотеку).

## Установка и запуск

1.  **Установите Python**
    Скачайте и установите Python с официального сайта: [python.org](https://www.python.org/downloads/).
    При установке (особенно на Windows) поставьте галочку «Add Python to PATH».

2.  **Скачайте проект**
    Скачайте архив с файлами проекта или склонируйте репозиторий Git.

3.  **Запустите программу**
    *   Откройте терминал (командную строку) в папке с проектом.
    *   Выполните команду:
        ```bash
        python main.py
        ```
    *   Откроется главное окно приложения.

## Структура проекта

В папке проекта находятся следующие файлы:
*   `main.py` — основной исполняемый файл приложения.
*   `books.json` — файл базы данных, где хранятся все добавленные книги.
*   `README.md` — этот файл с инструкцией.
*   `.gitignore` — файл для исключения временных файлов из системы контроля версий.

## Как пользоваться программой

### 1. Добавление новой книги
1.  Перейдите на вкладку **«Добавить книгу»**.
2.  Заполните все поля:
    *   **Название:** введите название книги.
    *   **Автор:** введите имя автора.
    *   **Жанр:** укажите жанр (например, «Фантастика», «Детектив»).
    *   **Страниц:** введите целое положительное число.
3.  Нажмите кнопку **«Добавить книгу»**.
4.  Книга появится в списке на вкладке «Список книг» и будет сохранена в файл `books.json`.

### 2. Фильтрация списка книг
1.  Перейдите на вкладку **«Фильтр»**.
2.  Введите критерии поиска:
    *   **Жанр:** введите название жанра (например, «Фантастика»). Если поле пустое, фильтрация по жанру не применяется.
    *   **Страниц >:** введите минимальное количество страниц (например, `300`). Если поле пустое, фильтрация по страницам не применяется.
3.  Нажмите кнопку **«Применить фильтр»**.
4.  На вкладке «Список книг» отобразятся только те книги, которые соответствуют выбранным критериям.

### 3. Просмотр полного списка
На вкладке **«Список книг»** всегда отображается текущая выборка (либо все книги, либо отфильтрованные).

## Примеры использования (Тесты)

1.  **Тест: Добавление книги**
    *   **Действие:** Введите «Мастер и Маргарита», «Михаил Булгаков», «Классика», `400`. Нажмите «Добавить».
    *   **Ожидаемый результат:** Книга появилась в таблице.
2.  **Тест: Ошибка ввода (пустое поле)**
    *   **Действие:** Оставьте поле «Автор» пустым. Нажмите «Добавить».
    *   **Ожидаемый результат:** Появится окно с ошибкой: «Все поля должны быть заполнены».
3.  **Тест: Ошибка ввода (не число)**
    *   **Действие:** В поле «Страниц» введите «много». Нажмите «Добавить».
    *   **Ожидаемый результат:** Появится окно с ошибкой: «Количество страниц должно быть положительным числом».
4.  **Тест: Фильтрация по жанру**
    *   **Действие:** На вкладке «Фильтр» введите жанр «Фантастика». Нажмите «Применить фильтр».
    *   **Ожидаемый результат:** В таблице останутся только книги, у которых в поле «Жанр» указано «Фантастика».
5.  **Тест: Комбинированный фильтр**
    *   **Действие:** На вкладке «Фильтр» введите жанр «Детектив» и количество страниц > `500`. Нажмите «Применить фильтр».
    *   **Ожидаемый результат:** В таблице останутся только детективы объемом более 500 страниц.
