# Book Tracker

**Автор:** Кузнецова Ирина Евгеньевна
**Описание:** Простое GUI-приложение для учёта прочитанных книг с фильтрацией и сохранением данных в JSON.

## Требования

- Python 3.8+
- Библиотеки: `tkinter` (входит в стандартную библиотеку Python), `json`
  Дополнительные библиотеки не требуются.

## Запуск приложения

1. Убедитесь, что Python установлен.
2. Скачайте файлы проекта.
3. Запустите:
   ```
   python main.py
   ```
4. В открывшемся окне используйте вкладки для добавления книг и фильтрации списка.
5. Данные сохраняются в файл `books.json`.

## Примеры использования

- Добавьте книгу: «Война и мир», Лев Толстой, Жанр: Классика (1225 страниц).
- Отфильтруйте книги по жанру «Фантастика».
- Покажите только книги объёмом более 300 страниц.
