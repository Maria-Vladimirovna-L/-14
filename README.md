# 👤 GitHub User Finder

**Автор:** Кузнецова Ирина Евгеньевна
**Версия:** 1.0

## Описание программы

«GitHub User Finder» — это графическое приложение для поиска пользователей на платформе GitHub с помощью официального **GitHub API**. Ключевые возможности:
*   Поиск пользователей по логину или имени.
*   Отображение списка найденных пользователей.
*   Добавление пользователей в «Избранное» с сохранением данных локально.
*   Открытие профиля пользователя в браузере по двойному клику.
*   Просмотр списка избранных пользователей.

## Системные требования и установка

1.  **Python 3.8+**
2.  **Библиотеки:**
    *   `tkinter` (входит в стандартную библиотеку Python).
    *   `requests` (для работы с API).
3.  **Установка зависимостей:**
    ```bash
    pip install requests
    ```
4.  **GitHub API:**
    *   Приложение использует **публичный API**, который не требует личного токена (API Key) для базового поиска пользователей.
    *   Лимит запросов без авторизации: до 60 запросов в час.

## Как использовать API (GitHub Search API)

Приложение использует эндпоинт `GET /search/users`.
*   **Параметр:** `q` — строка запроса (логин, имя или email).
*   **Пример запроса:** `https://api.github.com/search/users?q=defunkt`
*   Ответ содержит массив `items` с данными о пользователях.
*   Документация: [docs.github.com](https://docs.github.com/en/rest/search/search?apiVersion=2022-11-28#search-users)

## Запуск приложения

1.  Убедитесь, что Python и библиотека `requests` установлены.
2.  Запустите скрипт из терминала:
    ```bash
    python main.py
    ```
3.  Введите логин или имя пользователя в поле поиска и нажмите кнопку «Поиск».
4.  Для добавления в избранное — кликните правой кнопкой мыши по пользователю в списке.
5.  Для перехода на профиль — дважды кликните по строке пользователя.
6.  Данные об избранных пользователях сохраняются в файл `favorites.json`.
